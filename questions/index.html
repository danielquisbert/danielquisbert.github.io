Domina GeoServer.

Instala, configura y despliega GeoServer 3 en un servidor Linux real. Publica capas desde PostGIS, diseña estilos cartográficos y conecta todo a un visor web. Curso práctico de nivel intermedio.

Qué aprenderás en este curso
🖥️
Instalación en Linux
Java 17, Tomcat 11 y despliegue del archivo WAR en Ubuntu o Debian desde cero.
🐳
Docker y contenedores
GeoServer + PostGIS con Docker Compose. Despliegue reproducible en cualquier entorno.
🗺️
Publicación de capas
Workspaces, Stores y Layers. Datos vectoriales, raster y grupos de capas.
🎨
Estilos SLD y CSS
Simbología por categoría, etiquetas, escala dependiente y estilos para raster.
🗄️
PostGIS como fuente
Importa datos, crea vistas SQL y publica capas directamente desde la base de datos.
🔒
Servicios OGC y seguridad
WMS, WFS, WMTS, filtros CQL y control de acceso por roles y workspace.
